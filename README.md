# SinnerGPT

Allora praticamente è tipo ChatGPT, ma con più tennis.

Built with SvelteKit + Tailwind CSS.

## Usage

```sh
bun dev          # dev server
bun run build    # production build
```

## Deploy

Copy the `build/` folder and `docker-compose.yml` to your server.

Edit the compose file as needed; if unsure, just uncomment the port mapping lines.
